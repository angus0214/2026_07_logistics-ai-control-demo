## Agent skills

### Issue tracker

Issues are tracked using GitHub Issues. External PRs are not treated as a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

The default label vocabulary is used (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

This repository uses a multi-context layout with a root `CONTEXT-MAP.md`. See `docs/agents/domain.md`.
